[ 웹 과제8 : JPA 연관관계 설계 ]
    주제 :
    조건 1: 3개의 엔티티를 설계하시오
        제품카테고리 : 카테고리번호, 카테고리명 , ex] 1. 커피류 / 2. 스무디류
        제품 : 제품번호, 제품명, 제품가격            ex ] 1. 아메리카노,1000 / 2, 딸기 스무디, 15--
        주문 : 주문번호, 주문개수                 ex ] 1,3 / 2,2

    조건2 : 3개의 엔티티를 양방향으로 구현하시오
        - 제품테이블에 제품카테고리 FK 포함
        - 주문테이블에 제품번호 FK 포함

    조건3 : 모든 엔티티는 BaseTime 클래스를 상속받으시오.
